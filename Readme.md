# ArticleTemplate
Custom post type for Typecho

## Usage
Download release and upload to /usr/plugins, be sure folder is `ArticleTemplate`

How to ouput the value of template:
```
<?php $this->template() ?>
```
or
```
<?php $template = $this->template ?>
```
PS:`$this` is a Widget_Archive object
